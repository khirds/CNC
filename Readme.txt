1. config_I2SO.yaml 	=> contains Machine Defination and configurations
2. macrocfg.json 	=> contains the macro information and settings for "testgcode"(s)
3. setwork.g 		=> contains G code that relocates the "work coordinates" of the machine to the center of the bed
4. testgcode_circle.g	=> contains G code to engrave a Circle
5. testgcode_rect.g	=> contains G code to engrave a Rectangle
6. testgcode_triangle.g	=> contains G code to engrave a Triangle
